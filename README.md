
<h1 align="center">📚 Minha biblioteca </h1>

Este repositório é um ``mapa(índice)`` para todos os meus projetos,
pesquisas, estudos, experimentos e conhecimento (basicamente qualquer coisa). A proposta é trazer organização e facilidade de acesso para todos os projetos no repositório.

### ✨Motivação ([bla bla bla...pula isso ⬇](#conteúdo) )
Ideias e pensamentos vem e vão em uma velocidade tão alta no dia a dia que as vezes ficamos grandes períodos de mente vazia simplesmente seguindo a rotina, não existe uma retensão, é um "oi" e logo em seguida "vou descer no proximo ponto" para quem sabe nunca mais um segundo encontro. Meio dramático mas essa é a essência da coisa.

Então, decidi encontrar alguma maneira de registrar esses encontros para possivelmente desenvolver esses "oi"s que ocorrem ocasionalmente na minha vida. 

- Redes sociais ? com certeza não (Textos gigantes quase acadêmicos e outros arquivos no meu perfil ? eu passo)

- Drive/Pastas no PC ? é tentador mas vai virar uma bagunça com o tempo (se eu quiser compartilhar vai ser meio chato também)

- Aplicativos de gestão/projetos ? O plano free sempre acaba em algum momento e com uma equipe de 1 só não vale a pena toda a burocracia da plataforma (já basta no trabalho)

Então lembrei que já uso bastante o Github no meu dia a dia e aqui estou. Na prática vou usar o Github não só para armazenar código e projetos mas para estruturar minhas ideias, conhecimentos e opiniões. 

### 🤔 Oque eu encontro por aqui ?

><s> **"Este Hub não é apenas uma coleção de códigos. É o registro contínuo </s>
><s> de uma jornada de aprendizado, disciplina e construção de </s>
><s> conhecimento."**</s> - texto bonito do GPD 🤖 

Literalmente qualquer coisa que ficar matutando na minha cabeça e eu decidir dar continuidade. Temas aleatórios que podem envolver várias áreas de estudo ou até nenhuma.

Na real, em grande parte, vai ser uma exploração temática considerando valores (os meus obviamente), questionamentos e bastante "brainstorm" - não faltando também dados reais da ciência moderna para não virar uma obra de ficção científica (se bem que podia virar uma também).

Não pretendo deixar parte profissional separada (até porque pretendo usar a mesma conta do Github para facilitar no dia a dia). Então:

- Se você é da area profissional (TI): Vai ter uma paradas meio malucas no meio, mas se quiser ler e me conhecer melhor seja bem vindo.

- Se você for uma pessoa normal: Pode ser que tenha uns códigos e outras coisas técnicas de Ti no meio, só passa reto.
